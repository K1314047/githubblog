海纳思挂在外置移动硬盘

挂载外置硬盘

1、第一次挂载硬盘需要格式化，输入下面命令一键格式并挂载，执行命令之前一定要把你硬盘的资料备份一下

format-disk.sh

2、然后输入以下命令查看是否挂载成功

ll /mnt/sda1

3、显示以下内容，说明挂载成功

然后调整存储位置到外置硬盘

cd /mnt/sda1  #进入挂载硬盘

mkdir -p /mnt/sda1/files            #新建files文件夹
mkdir -p /mnt/sda1/downloads        #新建downloads文件夹
mkdir -p /mnt/sda1/home             #新建home文件夹

5、然后输入以下命令赋权限

chmod -R 777 downloads
chmod -R 777 files
chmod -R 777 home

SMBA共享外置硬盘

1、输入以下命令进行编辑

vi /etc/samba/smb.conf

2、进入之后按下insert键，然后将光标移到

3、先复制两个一样的代码然后再修改成下面的代码

[downloads]
  path = /mnt/sda1/downloads
  read only = no
  guest ok =yes
  create mask =0777
  directory mask = 0777
  browseable = yes
[files]
  path = /mnt/sda1/files
  read only = no
  guest ok =yes
  create mask =0777
  directory mask = 0777
  browseable = yes
[home]
  path = /mnt/sda1/home
  read only = no
  guest ok =yes
  create mask =0777
  directory mask = 0777
  browseable = yes

4、修改完成上面的代码之后按esc（取消） 再输入：wq（退出）

5、退出之后输入重启smb命令

systemctl restart smbd

6、完成重启之后就可以在此电脑-网络位置添加外置硬盘了

输入：\海纳思ip\downloads

​ \海纳思ip\files

​ \海纳思ip\home

BT下载配置文件 下载地址修改

1、停止bt下载命令

/etc/init.d/transmission-daemon stop

2、输入以下命令进行编辑

vi /etc/transmission-daemon/settings.json

3、进入之后按下insert键，然后将光标移到下图然后修改成/mnt/sda1/downloads

4、修改完成上面的代码之后按esc（取消） 再输入：wq（退出）

5、退出之后输入开启bt下载命令

/etc/init.d/transmission-daemon start

开启之后回到海纳思首页点开BT下载，在最下方显示可用空间为你外置硬盘的容量就说明挂载成功了

直连下载配置文件下载地址修改

1、输入以下命令进行编辑

vi /usr/local/aria2/aria2.conf

2、进入之后按下insert键，然后将光标移到下图然后修改成/mnt/sda1/downloads

3、修改完成上面的代码之后按esc（取消） 再输入：wq（退出）

4、退出之后输入重启smb命令

systemctl restart aria2c.service

重启之后回到海纳思首页点开直连下载打开aria2设置-基本设置，查看下载路是否已更改到你的硬盘位置，显示硬盘位置说明就已经挂载成功了

本地网盘硬盘位置更改

进入本地网盘现在位置：

cd /var/www/html/files/

输入删除默认路劲命令

rm home

建立软连接到硬盘

ln -s /mnt/sda1/home
